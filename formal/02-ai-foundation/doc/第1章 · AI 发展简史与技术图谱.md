## 1.1 什么是人工智能

### AI 的定义

**人工智能（Artificial Intelligence, AI）** 是让机器模拟人类智能行为的技术。

```
人类智能                          人工智能
├── 感知（看、听）        →      计算机视觉、语音识别
├── 学习（经验积累）      →      机器学习
├── 推理（逻辑思考）      →      知识推理、规划
├── 理解（语言交流）      →      自然语言处理
└── 行动（执行任务）      →      机器人、智能代理
```

### 强 AI vs 弱 AI

| 类型 | 说明 | 现状 |
|------|------|------|
| **弱 AI（专用 AI）** | 在特定任务上超越人类 | 已实现（围棋、图像识别） |
| **强 AI（通用 AI）** | 具备人类级别的通用智能 | 尚未实现 |
| **超级 AI** | 远超人类的智能 | 假想阶段 |

> **当前的大语言模型属于弱 AI**，但在通用性上已有重大突破。

## 1.3 AI 技术栈全景图

### 层级关系

```
┌─────────────────────────────────────────────────────────┐
│                    人工智能 (AI)                        │

│  │              机器学习 (ML)                       │   │
│  │  从数据中学习规律，无需显式编程                    │   │
│  │  ┌─────────────────────────────────────────┐   │   │

│  │  │  使用深层神经网络学习特征表示              │   │   │
│  │  │  ┌─────────────────────────────────┐   │   │   │
│  │  │  │      大语言模型 (LLM)            │   │   │   │
│  │  │  │  基于 Transformer 的超大规模模型  │   │   │   │
│  │  │  └─────────────────────────────────┘   │   │   │
│  │  └─────────────────────────────────────────┘   │   │

```

### AI 技术分支

| 分支 | 任务 | 代表技术 |
|------|------|---------|
| **计算机视觉 (CV)** | 图像识别、目标检测 | CNN、YOLO、SAM |
| **自然语言处理 (NLP)** | 文本理解、对话生成 | Transformer、GPT |
| **语音技术** | 语音识别、语音合成 | Whisper、TTS |
| **推荐系统** | 个性化推荐 | 协同过滤、深度推荐 |
| **强化学习** | 决策与控制 | DQN、PPO |

### 部署方式

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **云端 API** | 即开即用、免维护 | 数据安全、成本累积 | 快速验证、中小规模 |
| **私有化部署** | 数据可控、定制化 | 成本高、需要运维 | 企业核心业务 |
| **边缘部署** | 低延迟、离线可用 | 能力受限 | 端侧智能 |

## 1.5 大模型为什么改变了一切

### 范式转变

```
传统 AI 开发                         大模型时代
─────────────────────────────────────────────────────
为每个任务训练专用模型     →      一个模型解决多种任务
需要大量标注数据           →      零样本/少样本学习
需要 ML 专业知识           →      用自然语言指令
开发周期长                 →      快速原型验证
能力上限明显               →      涌现出意想不到的能力
```

### 为什么大模型能力涌现

1. **规模效应** — 参数量从百万到万亿，质变
2. **海量数据** — 互联网级别的文本训练
3. **Transformer** — 高效处理长距离依赖
4. **算力突破** — GPU/TPU 集群训练
