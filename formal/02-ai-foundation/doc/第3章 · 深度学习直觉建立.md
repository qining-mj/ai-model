## 3.1 神经网络基础

### 从生物神经元到人工神经元

```
生物神经元                           人工神经元
                                    
    树突                                输入
      ↘                                  ↘
   ───●─── 轴突                    x₁ ──→ ╱─╲
      ↗                           x₂ ──→│ Σ │──→ 激活函数 ──→ 输出
    树突                           x₃ ──→ ╲─╱
                                          ↑
接收信号 → 处理 → 发送              加权求和 → 非线性变换 → 输出
```

### 感知机：最简单的神经网络

```python
# 单个神经元的计算
def 神经元(输入, 权重, 偏置):
    加权和 = sum(x * w for x, w in zip(输入, 权重)) + 偏置
    输出 = 激活函数(加权和)
    return 输出

# 数学表达
y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)
y = f(W·X + b)
```

### 激活函数：引入非线性

**为什么需要激活函数？**

没有激活函数，多层网络等于单层（线性变换的组合还是线性）

```
常见激活函数

Sigmoid: f(x) = 1/(1+e⁻ˣ)        ReLU: f(x) = max(0, x)
    1 ┤      ───────                  │      ╱
      │    ╱                          │    ╱
  0.5┤  ╱                             │  ╱
      │╱                              │╱
    0 └──────────                   0 └──────────
       -5    0    5                   -2   0   2

用于概率输出                        现代网络最常用
```

### 多层感知机（MLP）

```
输入层          隐藏层           输出层

 x₁ ──○───────○──────○
      ╲     ╱ ╲    ╱
 x₂ ──○───○─────○─────○──→ y
      ╱     ╲ ╱    ╲
 x₃ ──○───────○──────○

每层的每个神经元与下一层全连接
```

## 3.2 深度学习的"深"

### 浅层 vs 深层

```
浅层网络 (2-3层)                  深层网络 (数十到数百层)
                                 
输入 → 隐藏 → 输出                输入 → 层1 → 层2 → ... → 层N → 输出

学习简单模式                      学习层次化的复杂特征
```

### 特征层次

**以图像识别为例：**

```
输入图片
   ↓
第1层：检测边缘和纹理    ─────╲
   ↓                        ╲
第2层：检测形状和局部模式  ─────→ 从低级到高级
   ↓                        ╱    自动学习特征
第3层：检测物体部件      ─────╱
   ↓
第4层：识别完整物体

    边缘 → 眼睛/鼻子 → 脸部 → "这是一只猫"
```

### 表示学习

> **深度学习的核心能力：自动学习数据的有效表示**

```
传统机器学习                      深度学习
─────────────────────────────────────────────
人工设计特征                      自动学习特征
（特征工程）                      （表示学习）

图片 → 人工提取边缘/颜色 → 分类   图片 → 神经网络 → 分类
      ↑                               ↑
    需要专家知识                     端到端学习
```

## 3.3 为什么深度学习能工作

### 三驾马车

```
                深度学习成功
                    │
        ┌───────────┼───────────┐
        ↓           ↓           ↓
      数据        算力        算法
      
   互联网时代    GPU/TPU     反向传播
   海量数据      并行计算     优化算法
   众包标注      云计算       网络架构
```

### 数据的力量

| 时代 | 数据量 | 效果 |
|------|--------|------|
| 2000年前 | MB级 | 简单任务 |
| 2010年代 | GB级 | 图像分类突破 |
| 2020年代 | TB-PB级 | 大语言模型涌现 |

### 算力的飞跃

```
GPU 并行计算优势

CPU：串行处理                 GPU：并行处理
┌─┐                          ┌─┬─┬─┬─┬─┬─┬─┬─┐
│█│                          │█│█│█│█│█│█│█│█│
└─┘                          └─┴─┴─┴─┴─┴─┴─┴─┘
一个任务接一个                 同时处理多个任务

矩阵运算天然适合并行
深度学习 = 大量矩阵运算
```

### 预训练 + 微调范式

```
传统方式                         预训练 + 微调
─────────────────────────────────────────────────

每个任务都要大量数据              └→ 任务A（少量数据）
                                 └→ 任务B（少量数据）

```

## 3.4 主要网络架构一览

### CNN（卷积神经网络）

**图像处理之王**

```
输入图片                     特征图                    输出
┌───────────┐           ┌───────────┐
│           │  卷积核   │           │
│  图 像    │ ───────→  │ 特 征     │ ───→ 全连接 ───→ "猫"
│           │  池化     │           │
└───────────┘           └───────────┘

卷积：检测局部特征（边缘、纹理）
池化：降低维度，增强鲁棒性
```

**核心特点**：
- 局部连接（不是全连接）
- 权重共享（同一个卷积核扫描整张图）
- 平移不变性（猫在左边和右边都能识别）

### RNN/LSTM（循环神经网络）

**序列数据处理**

```
处理句子："我 爱 学习"

     ┌───┐     ┌───┐     ┌───┐
"我"→│ h₁│────→│ h₂│────→│ h₃│→ 输出
     └───┘     └───┘     └───┘
               ↑          ↑
             "爱"       "学习"

每一步都考虑之前的信息（记忆）
```

**问题**：长距离依赖难以学习（梯度消失）

**LSTM**：通过门控机制解决长距离依赖

### Transformer

**注意力就是一切**

```
输入序列的每个位置都可以"看到"其他所有位置

    我   爱   学习
    ↓    ↓    ↓
  ┌─────────────┐
  │  Self-     │
  │  Attention │  每个词都关注所有词
  └─────────────┘
    ↓    ↓    ↓
   输出  输出  输出
```

**核心优势**：
- 并行计算（不像 RNN 必须顺序）
- 长距离依赖建模能力强
- 成为现代大模型的基础架构

### 架构选型

| 数据类型 | 推荐架构 | 示例应用 |
|---------|---------|---------|
| 图像 | CNN | 图像分类、目标检测 |
| 序列（短） | RNN/LSTM | 情感分析、时序预测 |
| 文本（长） | Transformer | 语言模型、翻译 |
| 多模态 | Transformer | GPT-4o、Gemini |
