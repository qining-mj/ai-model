## Agent 核心概念

```
┌─────────────────────────────────────────────────────────┐
│                      AI Agent                           │
├─────────────────────────────────────────────────────────┤
│  感知 (Perception)  →  推理 (Reasoning)  →  行动 (Action)│
│       ↑                                          │      │
│       └──────────── 反馈 (Feedback) ─────────────┘      │
└─────────────────────────────────────────────────────────┘
```

---

## Agent vs Chain

| 特性 | Chain | Agent |
|------|-------|-------|
| 执行流程 | 预定义 | 动态决策 |
| 灵活性 | 低 | 高 |
| 工具使用 | 固定 | 按需 |
| 循环能力 | 无 | 有 |
