## 依赖安装

```bash
# 核心依赖
pip install langchain langchain-openai chromadb

# 文档处理
pip install pypdf docx2txt unstructured

# Web 服务
pip install fastapi uvicorn python-multipart

# CLI 工具
pip install typer rich

# 环境变量
pip install python-dotenv
```
