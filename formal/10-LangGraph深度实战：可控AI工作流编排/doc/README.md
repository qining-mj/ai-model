## LangGraph 核心概念

```
┌─────────────────────────────────────────────────────────┐
│                     LangGraph                           │
├─────────────────────────────────────────────────────────┤
│  State（状态）- 数据容器，贯穿整个流程                   │
│  Node（节点）- 处理逻辑，修改状态                        │
│  Edge（边）  - 连接节点，可带条件                        │
│  Graph（图） - 编排整体工作流                            │
└─────────────────────────────────────────────────────────┘
```

---

## LangGraph vs LCEL

| 特性 | LCEL | LangGraph |
|------|------|-----------|
| 执行模式 | 线性/DAG | 循环图 |
| 状态管理 | 隐式 | 显式 |
| 循环支持 | ❌ | ✅ |
| 检查点 | ❌ | ✅ |
| 人机协作 | 困难 | 原生支持 |
