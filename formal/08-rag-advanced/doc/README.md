# 模块 8：RAG 高级技术

> **学习目标**：深入掌握 RAG 系统的完整设计与优化

## 模块概述

本模块深入讲解 RAG（检索增强生成）技术，从架构设计到生产优化，帮助你构建高质量的知识问答系统。

## RAG 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    RAG 完整流程                          │
├─────────────────────────────────────────────────────────┤
│  离线索引: 文档 → 分块 → 向量化 → 存储                   │
├─────────────────────────────────────────────────────────┤
│  在线检索: 查询 → 检索 → 重排序 → 生成                   │
└─────────────────────────────────────────────────────────┘
```

---

## 核心技术选型

| 组件 | 推荐方案 | 备选 |
|------|---------|------|
| Embedding | OpenAI text-embedding-3 | BGE, E5 |
| 向量数据库 | Chroma (开发) / Milvus (生产) | Pinecone, Qdrant |
| 分块策略 | 递归分块 | 语义分块 |
| 检索策略 | 混合检索 + Rerank | MMR |
| LLM | GPT-4o-mini | Claude, 国产模型 |
