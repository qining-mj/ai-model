## 核心架构

### Transformer 结构

```
┌─────────────────┬─────────────────┐
│     编码器       │     解码器       │
├─────────────────┼─────────────────┤
│ Self-Attention  │ Masked Attention│
│       ↓         │       ↓        │
│  Add & Norm     │  Add & Norm    │
│       ↓         │       ↓        │
│                 │ Cross-Attention│
│       ↓         │       ↓        │
│    FFN          │     FFN        │
│       ↓         │       ↓        │
│  Add & Norm     │  Add & Norm    │
└─────────────────┴─────────────────┘
```

### GPT 演进

```
GPT-1 (2018)    GPT-2 (2019)    GPT-3 (2020)    ChatGPT (2022)   GPT-4 (2023)
    │               │               │                │               │
  117M            1.5B            175B            RLHF           多模态
预训练+微调      Zero-Shot      Few-Shot        人类对齐        推理增强
```
